# youtube-video-downloader

## About
This is a Flask api written in Python3 that allows you to
download any YouTube video for free.
Created by Andrej Vujić, on March 13th 2021.

## How to download a video?
Open ```https://yt-dwnldr.herokuapp.com/download?video_url=<YOUR_VIDEO_URL>```, and set the ```<YOUR_VIDEO_URL>```
to the url of the YouTube video you want to download.
